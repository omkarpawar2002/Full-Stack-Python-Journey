# Day 9 - Sets

## Overview

Learned the Set data structure in Python. Sets are unordered collections that store unique values and are commonly used for removing duplicates, fast membership testing, and performing mathematical set operations.

## Topics Covered

### Basic Concepts

* What is a Set
* Creating Sets
* Unique Values
* Set Characteristics
* Empty Set vs Empty Dictionary

### Set Methods

* `add()`
* `update()`
* `remove()`
* `discard()`
* `pop()`
* `clear()`

### Membership Operations

* `in`
* `not in`

### Useful Functions

* `len()`
* List to Set Conversion
* Set to List Conversion

### Set Operations

* Union
* Intersection
* Difference
* Symmetric Difference

### Advanced Topics

* `copy()`
* `issubset()`
* `issuperset()`
* `isdisjoint()`
* `frozenset`


## Common Use Cases

* Removing duplicates from data
* Fast membership checking
* Comparing collections of data
* Finding common or unique elements
* Data filtering and validation

## Skills Gained

* Creating and managing sets
* Adding and removing elements
* Performing set operations
* Converting between lists and sets
* Removing duplicates efficiently
* Working with subset and superset relationships
* Understanding immutable collections with `frozenset`
