# Variables and Data Types

## 📖 Overview

I learned one of the most fundamental concepts in Python: **Variables and Data Types**.

Variables are used to store data, while data types define the kind of data being stored. Understanding these concepts is essential because almost every Python program relies on variables to manage information.

---

## 🎯 Learning Objectives

By the end of this day, I was able to:

* Understand what variables are and why they are used
* Follow Python variable naming rules
* Differentiate between valid and invalid variable names
* Understand dynamic typing in Python
* Assign and reassign values to variables
* Use multiple assignment
* Follow constant naming conventions
* Identify and use different Python data types
* Check the type and identity of objects using `type()` and `id()`

## Data Types Overview

Python provides several built-in data types.

### Numeric Types

* `int`
* `float`
* `complex`

### Boolean Type

* `True`
* `False`

### Sequence Types

* `str`
* `list`
* `tuple`

### Mapping Type

* `dict`

### Set Type

* `set`

### None Type

* `None`

## 🔑 Key Takeaways

* Variables store data for later use.
* Python uses dynamic typing.
* Variable names must follow specific rules.
* Python supports multiple built-in data types.
* `type()` helps identify data types.
* `id()` helps identify object references.
* Constants are written in uppercase by convention.
* Understanding variables and data types is essential before learning input handling, conditions, loops, and functions.
