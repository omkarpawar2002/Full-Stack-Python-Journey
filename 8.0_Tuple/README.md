# Tuples

## Overview

Learned the Tuple data structure in Python and how it differs from Lists. Tuples are ordered, immutable collections used to store multiple values in a single variable.

## Topics Covered

* Creating Tuples
* Single Element Tuple
* Tuple Packing
* Tuple Unpacking
* Tuple Indexing
* Tuple Slicing
* Immutability
* Tuple Methods

  * `count()`
  * `index()`

## Key Concepts

* Tuples are created using `()`.
* Tuples can store multiple data types.
* Tuples maintain insertion order.
* Tuples allow duplicate values.
* Tuples are immutable (cannot be modified after creation).
* A single-element tuple requires a trailing comma.
* Packing combines multiple values into a tuple.
* Unpacking assigns tuple values to separate variables.
* Indexing starts from `0`.
* Negative indexing starts from `-1`.
* Slicing extracts a portion of a tuple.

## Comparison: List vs Tuple

| Feature              | List | Tuple |
| -------------------- | ---- | ----- |
| Syntax               | `[]` | `()`  |
| Mutable              | Yes  | No    |
| Ordered              | Yes  | Yes   |
| Duplicates Allowed   | Yes  | Yes   |
| Memory Efficient     | No   | Yes   |
| Modification Allowed | Yes  | No    |

## Skills Gained

* Creating and using tuples
* Accessing tuple elements
* Extracting data using slicing
* Packing and unpacking values
* Working with immutable data
* Using tuple methods effectively
